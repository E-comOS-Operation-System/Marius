# Marius - Network connection program and network card management
## How to use
You must go in to ctl , then you can continue to use Marius 
When you see this screen , its means you went to Marius now!
```
WELCOME TO MARIUS
VESION X 2025 , COPYTIGHT (C) 2025 Saladin5101

```
**The interface may differ between different versions.**
### Commands
We list some commands you can use :
```
connect -  connect to network
device - managment your network device
help - get help for Marius
exit - eixt this ctl
disconnect - disconnect to network
```
